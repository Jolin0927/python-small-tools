Day2: 文件持久化 TODO 待办工具
📚 核心知识点
- 掌握了 `open()` 函数的文件读写操作
  - 学会用 `with` 上下文管理器安全地读写文件，自动关闭资源
  - 理解了读写模式 `"r"`（读）和 `"w"`（写）的区别
- 熟练使用 `json.dump()` 和 `json.load()` 实现数据持久化
  - `json.dump()`：把 Python 列表/字典序列化为 JSON 格式，写入文件保存任务
  - `json.load()`：从 JSON 文件中读取数据并反序列化为 Python 对象，实现程序重启后任务不丢失
- 巩固了异常处理、用户交互和模块化编程的思路

 🛠️ 解决的问题
- 解决了程序关闭后任务数据丢失的痛点，实现了永久保存
- 处理了文件不存在、JSON 格式损坏等异常场景，提升了程序健壮性
- 学会了处理中文乱码问题，通过 `encoding="utf-8"` 和 `ensure_ascii=False` 保证中文正常读写
